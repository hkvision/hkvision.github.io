Introduction

Our Final Year Project aims at creating an AI Chat Bot that commuincates with Microsoft customers through online conversations in natural languages and guides them in an interactive and effective manner to recommend them with the most suitable product. The Bot also helps staff handle complaints from customers more efficiently. The main modules of this project include Bot Framework, Language Understanding Intelligent Service (LUIS), Machine Learning Studio, Text Translator and SQL Database. The final product is hosted on Skype and Facebook Messager. More details can be found from the documentations in the Progress Section of this page.
